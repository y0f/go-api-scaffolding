# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the
work. Harassment and personal attacks are not tolerated.

Report unacceptable behavior through GitHub's private vulnerability or
moderation channels for this repository. Maintainers will review and respond to
every report.
